# Enterprise Network Monitoring & Security Monitoring Lab

## Overview

This repository contains practical labs and a final project developed for the **Network Security Monitoring Systems** course. The project focuses on designing, deploying, and evaluating modern monitoring and security solutions for enterprise environments.

The work covers both **Infrastructure Monitoring (NMS)** and **Security Information and Event Management (SIEM)** approaches through hands-on implementation using open-source technologies.

## Objectives

* Understand the architecture and operation of modern monitoring systems.
* Deploy and configure enterprise monitoring solutions.
* Collect, analyze, and visualize system and security events.
* Detect abnormal activities and potential cyber attacks.
* Evaluate monitoring strategies for organizations of different sizes.
